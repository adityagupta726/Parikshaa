How To Use This Application

Create Test

The Tests section is where you add and edit questions .Only teacher has access to the create Test section. Create a test and then add questions directly into the test (you can provide as many options you want). These questions are also automatically added to the Questions Record section and then you have to click on create test button and Your test is ready!

Attempt Test

The attempt section is where students can attempt tests created by teachers.The unattempted and attempted test are shown on their dashboards. By clicking on unattempted tests they can attempt those test within a time duration. They can view previous and next questions by clicking on buttons and when test gets over they have to click on submit button and it will automatically ends if the time gets over.

Review Test

The review section is where students can see their attempted tests .They can view their answers and the correct ones and analyse their result .Both student and teacher has access to the review section.Teachers can view their created test and questions by clicking on review test on their dashboard.
